# 九州争鼎 — VIP 体系设计文档 v2.0

## 一、设计原则

| 原则 | 说明 |
|------|------|
| **非Pay-to-Win** | VIP 提供便利性和效率，不直接提升战斗数值 |
| **渐进付费** | 低等级VIP即可获得核心体验，高等级提供边际收益递减的福利 |
| **双轨制** | 累充等级（永久）+ 订阅等级（限时）并行 |
| **活跃挂钩** | VIP每日奖励需登录领取，驱动日活 |

---

## 二、VIP 双轨体系

### 2.1 累充等级（荣誉等级·永久）

通过累计充值钻石获得VIP经验，等级永久保留。

```
VIP经验 = 充值钻石数量 × 1
```

| VIP等级 | 累充钻石 | 累计金额(¥) | 核心特权 |
|---------|---------|------------|---------|
| VIP0 | 0 | ¥0 | 基础功能 |
| VIP1 | 500 | ¥50 | 购买体力+2次/日，背包+20格 |
| VIP2 | 2,000 | ¥200 | 扫荡次数+5/日，行军队列+1 |
| VIP3 | 5,000 | ¥500 | 每日抽卡折扣(9折)，联盟商城额外商品 |
| VIP4 | 12,000 | ¥1,200 | 世界聊天红色字体，每日免费单抽×1 |
| VIP5 | 25,000 | ¥2,500 | 行军队列+1(共+2)，扫荡+10/日 |
| VIP6 | 50,000 | ¥5,000 | 城池税收+10%，每日免费十连×1(含保底) |
| VIP7 | 100,000 | ¥10,000 | 私人城池改名，专属VIP头像框 |
| VIP8 | 200,000 | ¥20,000 | 专属坐骑皮肤，行军速度+10% |
| VIP9 | 500,000 | ¥50,000 | 专属城池皮肤，全服广播权 |
| VIP10 | 1,000,000 | ¥100,000 | 荣誉雕像(主城)，专属称号"九州之主" |

> **设计依据**: VIP0→5 覆盖90%付费用户（小额付费），VIP6→10 为大R专属（ARPU主力）

## 三、VIP 经济模型测算

### 3.1 用户分层 ARPU 模型

| 用户类型 | 占比 | 月均消费(¥) | 核心付费点 | LTV(12月) |
|---------|------|-----------|-----------|----------|
| 免费用户 | 60% | ¥0 | — | ¥0 |
| 小R(小鱼) | 25% | ¥30~100 | 月卡+首充 | ¥600 |
| 中R(海豚) | 10% | ¥200~800 | 月卡+礼包+少量抽卡 | ¥6,000 |
| 大R(鲸鱼) | 4% | ¥2,000~10,000 | 深度抽卡+战令+VIP | ¥60,000 |
| 超R(巨鲸) | 1% | ¥20,000+ | 全购+冲榜+联盟战 | ¥200,000+ |

### 3.2 整体 ARPU 计算

```
ARPU = 0.6×0 + 0.25×65 + 0.10×500 + 0.04×6000 + 0.01×30000
     = ¥0 + ¥16.25 + ¥50 + ¥240 + ¥300
     = ¥606.25/月

千人收入 = ¥606,250/月
```
