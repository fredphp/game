## 二、卡池矩阵设计

### 2.1 常驻卡池 — 「英杰招募」

- 常驻开放，消耗 **招募令(免费)** 或 **钻石(付费)**
- SSR池: 全部常驻SSR角色（约30张）
- 新手30抽必出SSR（新手福利，每个账号限1次）
- 每日免费单抽 ×1

### 2.2 限定卡池 — 「乱世英雄」

- 每 **28天** 轮换一期，含1~2张限定UP角色
- 限定角色 **当期结束后6个月内不入常驻池**
- 保底计数独立于常驻池
- 开放期间十连额外赠送「限定碎片×5」

### 2.3 阵营卡池 — 「天下大势」

- 聚焦单一阵营（魏/蜀/吴/群），每期14天轮换
- UP概率提升至 **75%**（常驻50%）
- 阵营相关材料掉率+30%
- 抽满 **200抽** 解锁「阵营精通」称号

### 2.4 武器池 — 「神兵利器」

- 独立于角色池，产出武器/装备
- 概率: UR 3% / SSR 15% / SR 82%
- 硬保底: 50抽出UR
- 武器可强化至+15

### 2.5 新手池 — 「初出茅庐」

- 仅新账号（注册7天内）可抽
- 首次十连 **半价**（900钻石）
- 保底: 10抽必出SR，30抽必出SSR
- 结束后自动转入常驻池

## 五、付费心理学设计

### 5.1 锚定效应
- 十连标价 ¥180，月卡 ¥30 看似"极大折扣"（实际月卡需30天领完）
- ¥6 首充显示"价值 ¥180"（原价10抽 vs 现价1抽）

### 5.2 沉没成本
- 限定池大保底机制：已抽60+抽的玩家极难放弃（"再20抽就保底了"）
- 成长基金：已投入¥30，不升级等于"浪费"

### 5.3 稀缺性
- 限定角色"绝版6个月"制造FOMO
- 每周限量礼包制造紧迫感
- 战令限时赛季

### 5.4 社交攀比
- VIP等级头像框全服可见
- 限定皮肤/称号
- 联盟排行榜刺激消费
