## 二、新手引导（D0~D2）— 首日体验设计

### 2.1 前15分钟体验流

```
[0min]  进入游戏 → 选择阵营(魏/蜀/吴/群) → 个性化初始体验

[3min]  强制教程: 基础战斗(1场自动战斗) → 理解核心玩法
[6min]  获得首张SR卡 → 激活收集欲望
[8min]  主城自由探索 → 开放资源建筑
[10min] 首次抽卡(免费十连) → 高亮时刻(必出1SR)
[12min] 弹窗: "¥1超值礼包" → 首次付费破冰点
[15min] 开放世界地图 → 看到九州全景，建立长期目标
```

## 三、7日留存设计（D1~D7）— 短期目标锁定期

### 3.1 七日签到表

| 天数 | 免费奖励 | 付费奖励(¥6七日卡) | 心理设计 |
|------|---------|------------------|---------|
| Day1 | SR自选券×1 | 额外SR自选券×1 + 钻石×100 | **立即获得感** |
| Day2 | 体力×120 | 体力×120 + 金币×5万 | 习惯形成 |
| Day3 | 十连券×1 | 额外十连券×1 + 扫荡券×10 | **中期奖励** |
| Day4 | 金币×5万 + 经验书×20 | 突破石×30 + 金币×5万 | 平稳期 |
| Day5 | SSR碎片×15 | 额外SSR碎片×15 + 限定头像框 | 接近目标 |
| Day6 | 体力×120 + 十连券×1 | 十连券×1 + 竞技币×500 | **冲刺期待** |
| **Day7** | **SSR自选券×1** | **额外SSR自选券×1 + 钻石×500** | **终极奖励！** |

> **7日签到总价值**: 免费=SSR自选×2 + 十连×2 + 碎片×15 ≈ ¥300+ | 付费(¥6)=再翻倍

### 3.2 七日成长任务链

```
Day 1: "初出茅庐"
├── 完成3场战斗 → 钻石×50
├── 升级1张卡到Lv.10 → 突破石×10
└── 完成全部日常 → 金币×2万
    └── [奖励] SR碎片×5

├── 抽卡3次(含免费) → 招募令×3
├── 组建5人战队 → 钻石×50
└── 通关PVE第3章 → 体力×60
    └── [奖励] 金币×5万 + 经验书×10

Day 3: "开疆拓土"
├── 占领2座城池 → 金币×3万
├── 完成1次行军 → 行军加速券×2
└── 加入联盟 → 钻石×100
    └── [奖励] 十连券×1

Day 4: "磨砺精兵"
├── 升级3张卡到Lv.15 → 突破石×20
├── 使用1次扫荡 → 扫荡券×5
└── 竞技场挑战3次 → 竞技币×200
    └── [奖励] SR碎片×10

├── 捐献资源给联盟 → 联盟贡献×500

├── 查看联盟成员列表 → 金币×2万
└── 与盟友组队1次 → SSR碎片×5
    └── [奖励] 限定头像框 + 钻石×100

├── 完成爬塔30层 → 钻石×100

    └── [奖励] 十连券×1

├── 完成前6天所有任务 → 钻石×200
├── 占领5座城池 → 金币×10万
├── 战力达到12000 → SSR碎片×20
└── 购买首充 → [首充奖励翻倍]
    └── [终极奖励] SSR自选券×1 + 限定称号"初露锋芒"
```

### 4.4 社交留存机制

```
社交关系强度 = f(联盟绑定, 好友互动, 协作记忆)

联盟绑定:
├── 联盟贡献不可转移 → 离开=失去所有贡献
├── 联盟城池共同所有 → 离开失去领土归属
├── 联盟战争CD → 加入新联盟有7天战事CD
└── 联盟仓库 → 捐献的物资归联盟所有

好友互动:
├── 每日互赠体力 → 双方各得30体力
├── 好友助战 → 副本可邀请好友助战
├── 好友排行 → 谁都不想垫底
└── 组队副本 → 3人组队, 奖励+30%

协作记忆:
├── 联盟战共同经历 → "我们曾一起守住XX城"
├── 联盟聊天 → 情感纽带
└── 联盟日志 → 历史记忆
```

## 五、付费留存策略

### 5.1 月卡留存模型

```
月卡用户留存链:

Day 0  购买月卡(¥30) ──→ ¥30沉没成本
  │
Day 1  领取每日100钻 ──→ "不能浪费"
  │
Day 7  已领取700钻 ──→ 价值¥70, 已回本
  │
Day 15 续费提醒 ──→ "还有15天到期"
  │
Day 25 到期预警 ──→ "还有5天, 不续就亏了"
  │
Day 30 到期 → 续费(¥30) 或 升级(¥68) ──→ 新的30天循环
```

### 5.2 沉没成本留存

| 沉没成本类型 | 累积速度 | 离开损失感 |
|------------|---------|-----------|
| 月卡未领钻石 | 100钻/日 | 中 |
| 战令未领等级 | ~2级/日 | 高 |
| 联盟贡献值 | 变速增长 | 高 |
| 卡牌养成进度 | 变速增长 | 极高 |
| 竞技段位 | 赛季重置 | 中 |
| 限定卡/皮肤 | 不可再生 | 极高 |

## 六、流失预警与召回

### 6.1 流失预警模型

```
流失概率 = f(登录频率↓, 游戏时长↓, 付费频率↓, 社交活跃度↓)

预警等级:
  🟡 黄色预警 (流失概率 30~50%):
    - 连续2天未登录
    - 触发: 推送通知 + 50钻召回邮件

  🟠 橙色预警 (流失概率 50~70%):
    - 连续4天未登录
    - 触发: 推送 + 200钻 + 好友代领提醒

  🔴 红色预警 (流失概率 70~90%):
    - 连续7天未登录
    - 触发: 推送 + 十连券×1 + 限定召回礼包(¥6超值)

  ⚫ 黑色预警 (流失概率 >90%):
    - 连续30天未登录
    - 触发: 回归活动入口 + 3张十连 + SSR自选
```

### 6.2 回归活动设计 — 「英雄归来」

| 回归天数 | 回归礼包内容 | 价格 | 限购 |
|---------|------------|------|------|
| 3~7天 | 钻石×500 + 体力×240 + 突破石×30 | ¥1 | 1次 |
| 8~14天 | 十连券×2 + SSR碎片×20 + 金币×10万 | ¥6 | 1次 |
| 15~30天 | 十连券×3 + SSR自选券×1 + 限定头像框 | ¥30 | 1次 |
| 30天+ | 十连券×5 + SSR自选券×2 + 限定皮肤 + 钻石×2000 | ¥68 | 1次 |
